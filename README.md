# Dashboard_con_KPI
Dashboard de Power BI enfocado en la medición de KPIs de una encuesta de satisfacción. El proceso incluyó la recolección, limpieza y transformación de datos (Power Query). Implementé la lógica de negocio y cálculos para generar visualizaciones robustas y obtener insights valiosos para el rendimiento y la toma de decisiones
